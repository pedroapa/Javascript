01. Crie uma função que recebe um array de 2 números e coloque eles em ordem crescente. Não use o método sort.

Pesquisei como mandar um vetor para dentro de uma função, vou ver se consigo fazer este exercicio.

Exemplo de como mandar: 

const arr = [1, 2, 3]

function sum(n1, n2, n3) {
    return n1 + n2 + n3;
}
console.log(sum(...arr)) // 6
